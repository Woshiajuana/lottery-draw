
# 抽奖事件

> 数据格式，前端上传给后台服务
> 事件名：directiveEvent

```
data
├── scene                                   // 场景值，字符串，  0001：签到   0002：奖品抽奖   0003：随机大抽奖
├── type                                    // 操作类型，字符串，0：停止  1：开始  2：展示
├── title                                   // 标题名称
├── number                                  // 抽奖人数

```

> 数据格式，后台服务给前端
> 事件名：directiveEvent

```
data
├── code                                    // 操作标志  字符串， 0000：成功   9999：失败
├── message                                 // 操作提示信息，字符串
├── data                                    // 数据
|   ├── scene                               // 场景值，字符串，  0001：签到   0002：奖品抽奖   0003：随机大抽奖
|   ├── type                                // 操作类型，字符串，0：停止  1：开始  2：展示
|   ├── title                               // 标题名称
|   ├── number                              // 抽奖人数

```
